# 🚀 AI Resume Analyzer

An intelligent web application that analyzes resumes, extracts key skills, and provides an instant ATS-style score with improvement insights.

Built to help candidates evaluate their resumes before applying for jobs.

---

## 🌟 Features

* 📄 Upload resume in PDF format
* 🧠 Automatic skill extraction
* 📊 Resume scoring system (0–100)
* ✅ Detected skills display
* ❌ Missing skills identification
* 🎨 Modern responsive UI
* ⚡ Fast and lightweight Flask backend

---

## 🛠️ Tech Stack

**Frontend**

* HTML5
* CSS3

**Backend**

* Python
* Flask

**Libraries Used**

* PyPDF2 (PDF text extraction)
* Flask (web framework)

---

## 🧩 How It Works

1. User uploads a resume (PDF).
2. Backend extracts text from the resume.
3. System matches keywords against a predefined skills database.
4. Resume score is calculated.
5. Results are displayed with:

   * Skills found
   * Missing skills
   * Overall score

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/nupursingh10/ai-resume-analyzer.git
cd ai-resume-analyzer
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash

```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000
```
